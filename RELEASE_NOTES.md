# 远程桌面连接客户端 - 发布版本 1.0

## 概述
远程桌面连接客户端是一个轻量级的 Windows 应用程序，允许您通过 IP 地址和端口连接到远程计算机。

## 包含的文件
- RemoteDesktopClient.exe - 命令行版本
- RemoteDesktopClientGUI.exe - GUI 版本（推荐使用）
- README.md - 使用说明
- LICENSE - 许可证

## 功能
1. 通过 IP 地址和指定端口连接远程桌面
2. 端口连通性测试
3. 使用 Windows 原生的远程桌面连接功能
4. 支持自定义 RDP 端口

## 使用方法
1. 双击运行 RemoteDesktopClientGUI.exe
2. 程序会显示消息框提示您输入连接信息
3. 应用程序将测试端口连通性
4. 如果端口可达，将启动远程桌面连接

## 系统要求
- Windows 7/8/10/11
- 远程计算机启用远程桌面服务
- 网络连通性

## 版本信息
- 版本: 1.0
- 发布日期: 2026年2月7日
- 平台: Windows x64/x86

## 故障排除
- 如果无法连接，请确保目标计算机的防火墙允许远程桌面连接
- 确认 IP 地址和端口号正确
- 确保目标计算机已启用远程桌面服务
