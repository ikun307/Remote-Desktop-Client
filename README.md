# 远程桌面连接客户端

这是一个用 Go 语言开发的远程桌面连接客户端，支持通过指定 IP 地址和端口连接到远程计算机。

## 功能特性

- 通过 IP 地址和端口连接远程桌面
- 保存和管理多个远程连接配置
- 支持自定义端口（默认为 3389）
- 用户友好的图形界面
- 支持全屏模式

## 系统要求

  - Windows 10/11
  - Go 1.19 或更高版本
  - 目标计算机启用了远程桌面服务

## 安装和运行

   1. 确保已安装 Go 语言环境 (1.19 或更高版本)
   2. 下载或克隆此项目
   3. 在项目目录中运行以下命令安装依赖：
        ```
        go mod tidy
        ```
   4. 编译并运行应用程序：
        ```
        go run main.go
        ```
   5. 或者编译为可执行文件：
        ```
        go build -ldflags="-H windowsgui" -o RemoteDesktopClient.exe .
        ```
## 使用说明

  1. 运行应用程序
  2. 点击"新建连接"按钮创建新的远程桌面连接
  3. 输入连接名称、IP 地址、端口号、用户名等信息
  4. 点击"连接"按钮建立远程桌面连接
  5. 或者从连接列表中选择一个连接，点击"连接"按钮

## 技术架构

  - GUI: 使用 Walk 库实现 Windows 原生界面
  - 远程桌面连接: 使用 Windows 自带的 mstsc 命令
  - 配置管理: 内存中管理连接配置

## 注意事项

  - 此软件仅支持 Windows 系统
  - 目标计算机必须启用远程桌面服务
  - 需要正确的凭据才能连接到远程计算机
  - 防火墙必须允许远程桌面连接（默认端口 3389）
